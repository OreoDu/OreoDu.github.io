Part. 4 Language structure

We put these language elements together to form different statements so that we can solve a certain problem. The way we organize the statements can be divided into different types of structures in order to make our program more effective. Generally, there are three types: sequential structure, conditional structure and iteration.

Python uses indentation to set the hierarchy of different compound statements instead of using{}. If the continuous statements maintains the same indentation then they belong to the same code block, which is equivalent to one specific function.

Ⅰ. Conditional structure
Sometimes, we have to separate our solutions into different branches and change the actions of the program according to some conditions. In python, we use if statement(use the keywords if, else, elif) to implement it.

· Chained conditionals: (two branches or more)

# calculate the payment
try:
    fh = float(input('Enter the hour: '))
    fr = float(input('Enter the rate: '))
    if fh<0 :
        print('Pleae enter the right number')
    elif fh>=40 :
        pay = (fh-40)*fr*1.5+40*fr
    else :
        pay = fh*fr
    print('Pay: ', pay)
except:
    print('Pease enter numeric input!')

elif: one of the conditions, else: catch the exceptions
(If more than one condition is true, only the first true branch executes.)

· Nested conditionals: 

if fh<0 :
    print('Pleae enter the right number')
else : if fh>=40 :
           pay = (fh-40)*fr*1.5+40*fr
       else :
           pay = fh*fr

Usually, we use chained conditionals rather than nested conditionals or simplify it with operators. “Flat is better than nested”. Because the nested structure will have a serious impact on the readability of code.

Ⅱ. Iteration

Repetitive tasks are the main work for computers. Python provides the while statement and for statement to let the computers do the tasks in a more efficient way.

· while statement (indefinite loops):

# output the max and min
smallest = None
largest = None
while True:
    num = input('Enter the num: ')
    if num == 'Done':
        break
    try:
        num = int(num)
    except:
        print('Invalid input!')
        continue
    if smallest is None or num<smallest:
        smallest = num
    if largest is None or num>largest:
        largest = num
print('The smallest:',smallest,'The largest:',largest)

break: If the condition is met, the program will jump out of the loop.

continue: Skip to the next iteration without finishing the body of the loop for the current iteration.

· for statement (definite loops)

When we want to loop through a set of words, the lines in a file, a list of numbers and so on, we can construct a definite loop using a for statement. 

# Multiplication table
for i in range(1, 10):
    for j in range(1, i + 1):
        print('%d*%d=%d' % (i, j, i * j), end='\t')
    print()

range(101)A sequence of integers from 0 to 100 can be generated.
range(1, 100)A sequence of integers from 1 to 99 can be generated.
range(1, 100, 2)An odd sequence of 1 to 99 can be generated and 2 is the the increment of the sequence of values (step size).

friends = ['oreo', 'sermantha', 'Damon']
for friend in friends:
    print('Happy New Year!',friend)

Part. Bug

Program should be written precisely otherwise the deviations and mistakes in your code will cause the Python to stop and can’t work out the final result. There are three main types of errors that you may encounter during.

· Syntax errors:

A syntax error means that you have violated the “grammar” rules of Python.

· Logic errors:

A mistake in the order of the statements or in how the statements relate to one another.

· Semantic errors:

The program is perfectly correct but it does not do what you intended for it to do.

Ex.5: Find the perfect number within 10,000 

import math

for num in range(1, 10000):
    result = 0
    for factor in range(1, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            result += factor
            if factor > 1 and num // factor != factor:
                result += num // factor
    if result == num:
        print(num)

Ex.6: Output all prime numbers within 100

import math

for num in range(2, 100):
    is_prime = True
    for factor in range(2, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=' ')
