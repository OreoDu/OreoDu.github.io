    There are two ways that you run your code.

    • · Write the script in the text editor and execute it in the command window or just type commands into the Python interpreter.

      python xxx.py

    • · Write and run the code in a interactive programming tool (such as IPython).

      pip install ipython

    How do the programs work inside the computer?

    First, let’s have a quick look at the simplified computer structure.

    (from Python for Everybody)

    When we finish our code and store it in the secondary memory , the Python interpreter will parses the source code and interprets the instructions and the complier will translate the high-level source code into machine language and put the resulting machine language into a file for later execution.

    Then when we execute the program, it will be stored in the main memory temporarily. The computer will retrieve the data form the input devices or the network, get the instructions form the main memory and do the computations in the center processing unit(CPU) with a high speed(billion times per second). In the end, the computer will send the results to the output devices.
