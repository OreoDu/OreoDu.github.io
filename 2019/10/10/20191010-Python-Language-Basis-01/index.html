Part.2 How do the programs work inside the computer?

First, let's have a quick look at the simplified computer structure (from Python for Everybody)

When we finish our code and store it in the secondary memory, the Python interpreter will parses the source code and interprets the instructions and the complier will translate the high-level source code into machine language and put the resulting machine language into a file for later execution.

Then when we execute the program, it will be stored in the main memory temporarily. The computer will retrieve the data form the input devices or the network, get the instructions form the main memory and do the computations in the center processing unit(CPU) with a high speed(billion times per second). In the end, the computer will send the results to the output devices.

Part.3 Language element

A language must have its own vocabulary and grammar. In the vocabulary of Python, basically, we have Reserved words (keywords), Variable, constant(numbers,letters,strings) and operators.

Ⅰ. Reserved words (keywords): Special identifiers with specific meanings and functions in python that can't be used by programmer.

(from Python for Everybody)

Ⅱ. Variable: · Types : numbers: integer(can use different radfixes: 0b100,0o100,100,0x100) float(sometimes use scientific notation: 1.234e2) complex number(ex: 3+2j) string(be enclosed in single or double quotation marks) Boolean(True, False)

We can use type() function to get the type of the variables.

a = 1.234e6
print(type(a)) #<class 'float'>

Variable types can be converted using functions built in Python. · int(): Convert a number or string to an integer and specify a radix. · float(): Convert a string to a floating point number. · str(): Convert the specified object to a string and specifies the encoding. · chr(): Convert the corresponding encoding (integer) to the string (a character). · ord(): Convert a string (a character) to the corresponding encoding (integer).

· Name rules: (usually use a mnemonic name) 1.Combination of numbers, letters, underscore (can not start with a number) 2.Case sensitive 3.Can not use reserved words 4.PEP 8 ( Style Guide for Python Code )

Part. 4 Language structure

We put these language elements together to form different statements so that we can solve a certain problem. The way we organize the statements can be divided into different types of structures in order to make our program more effective. Generally, there are three types: sequential structure, conditional structure and iteration.

Python uses indentation to set the hierarchy of different compound statements instead of using{}. If the continuous statements maintains the same indentation then they belong to the same code block, which is equivalent to one specific function.

Ⅰ. Conditional structure Sometimes, we have to separate our solutions into different branches and change the actions of the program according to some conditions. In python, we use if statement(use the keywords if, else, elif) to implement it.

· Chained conditionals: (two branches or more)

# calculate the payment
try:
    fh = float(input('Enter the hour: '))
    fr = float(input('Enter the rate: '))
    if fh<0 :
        print('Pleae enter the right number')
    elif fh>=40 :
        pay = (fh-40)*fr*1.5+40*fr
    else :
        pay = fh*fr
    print('Pay: ', pay)
except:
    print('Pease enter numeric input!')

elif: one of the conditions, else: catch the exceptions (If more than one condition is true, only the first true branch executes.)

· Nested conditionals:

if fh<0 :
    print('Pleae enter the right number')
else : if fh>=40 :
           pay = (fh-40)*fr*1.5+40*fr
       else :
           pay = fh*fr

Usually, we use chained conditionals rather than nested conditionals or simplify it with operators. "Flat is better than nested". Because the nested structure will have a serious impact on the readability of code.

Ⅱ. Iteration

Repetitive tasks are the main work for computers. Python provides the while statement and for statement to let the computers do the tasks in a more efficient way.

· while statement (indefinite loops):

# output the max and min
smallest = None
largest = None
while True:
    num = input('Enter the num: ')
    if num == 'Done':
        break
    try:
        num = int(num)
    except:
        print('Invalid input!')
        continue
    if smallest is None or num<smallest:
        smallest = num
    if largest is None or num>largest:
        largest = num
print('The smallest:',smallest,'The largest:',largest)

break: If the condition is met, the program will jump out of the loop.

continue: Skip to the next iteration without finishing the body of the loop for the current iteration.

· for statement (definite loops)

When we want to loop through a set of words, the lines in a file, a list of numbers and so on, we can construct a definite loop using a for statement.

# Multiplication table
for i in range(1, 10):
    for j in range(1, i + 1):
        print('%d*%d=%d' % (i, j, i * j), end='\t')
    print()

range(101)A sequence of integers from 0 to 100 can be generated. range(1, 100)A sequence of integers from 1 to 99 can be generated. range(1, 100, 2)An odd sequence of 1 to 99 can be generated and 2 is the the increment of the sequence of values (step size).

friends = ['oreo', 'sermantha', 'Damon']
for friend in friends:
    print('Happy New Year!',friend)

Ⅲ. Functions and Modules

In order to write high-quality code, we should eliminate the repetitive code which is doing the same type of operations. We can encapsulate those repetitive codes into a function module called "function" so that we "call" the "function" when we need it. Generally, function is a named and reusable sequence of statements that takes arguments as input, performs a computation and return results. (The naming rules of function are consistent with the naming rules of variables. )

# How to define a function? Example:calculate the factorial
def factorial(num):
    result =  1 
    for n in range (1,num+1):
        result *= n
    return result

Actually, factorial function is a built-in function in python, we can use such kind of functions directly without defining them.

· Function argument In Python, function arguments can have default values and also support for variable arguments which is quite different from other languages.

from random import randint

def roll_dice(n=2): # The default value is 2.
    total =  0 
    for _ in range(n):
        Total += randint(1,6)
     return total
# '*' indicates that args is a variable parameter. The type of the arg is tuple.
def add(*args): 
    Total =  0 
    for val in args:
        Total += val
    return total

print(roll_dice())
print(roll_dice(6))
print(add(2,45,23,44))

Python looks for a variable in the order of "local scope", "nested scope", "global scope" and "built-in scope".

def foo():
    b =  ' hello ' # b belongs to nested scope
    def bar ():
        c =  True # c belongs to local scope as a local variable
 
if  __name__  ==  ' __main__ ' :
    a =  100  # a belons to global scope as a global variable
    foo()

Some identifiers like input,print and int are built inside the python which belong to built-in scope. The local variables can only be used inside the function it belongs to, but we can access the global variables anywhere we want to. However, if we want to modify the value of the global variables inside a specific function, we should add global before the variables.

def  foo ():
    global a
    a =  200 
    print (a)   # 200

If  __name__  ==  ' __main__ ' :
    a =  100
    foo()
    Print (a)   # 200

In the actual development process, we should reduce the use of global variables due to its wide scope. Unexpected modifications may occur and also the memory occupied by variables are unable to be recycled.

· Module Maybe sometimes, we are faced with naming conflicts. When this happens, the latter definition will override the previous definition, which means that only one of the two functions with the same name is actually executed. At such a moment, it is better to define the function with the same name in different files. In Python, every file represents a module. We can use the import to import the specified module to distinguish different functions.

module1.py

def foo():
     print ('hello, world!' )

module2.py

def foo():
     print ('goodbye, world!')

test.py

import module1 as m1
import module2 as m2
m1.foo() #hello, world!
m2.foo() #goodbye, world!

However, besides the functions defined in the module we import, if there are some other executable codes, it'd be better to put those codes into the condition shown below so that the python interpreter will only execute them when the module is run directly and will not execute them when we import the module into other files.

def foo():
    print ('goodbye, world!')
# __name__ is an implicit variable in Python that represents the name of the module.
if  __name__ == ' __main__ ' :
    print ('hello, world!')
    foo()

· Syntax errors:
> A syntax error means that you have violated the “grammar” rules of Python.

· Logic errors:

A mistake in the order of the statements or in how the statements relate to one another.

Ex.5: Find the perfect number within 10,000

import math

for num in range(1, 10000):
    result = 0
    for factor in range(1, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            result += factor
            if factor > 1 and num // factor != factor:
                result += num // factor
    if result == num:
        print(num)

Ex.6: Output all prime numbers within 100

import math

for num in range(2, 100):
    is_prime = True
    for factor in range(2, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=' ')

Ex.7: Write a program to find all the palindrome prime within 10000.

import math

def prime(arg):
    is_prime = True
    for i in range(2,int(math.sqrt(arg))+1):
        if arg % i == 0 :
            is_prime = False
            break
    return is_prime

def palindrome(arg):
    result = 0
    number = arg
    while arg > 0:
        result = arg % 10 + result*10
        arg //= 10
    return result == number

exist = False
for num in range(10, 10000):
    if prime(num) and palindrome(num):
        print('%d is a palindrome prime!' % (num))
        exist = True
if exist is False :
    print('There is no palindrome prime!')
